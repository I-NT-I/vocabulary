Zur Gestaltung und Ideen sammlung wurde in diesem Projekt Figma genutzt. Figma ermöglicht es Designern eine visuelle Ideen sammlung zu erstellen, die den Programmierern erleichtert, mit dem Programmieren zu starten, dabei wird kein Code erstellt, sondern nur eine Ansammlung von Bildern. Die Aufgabe liegt jetzt bei den Programmierern, diese Bilder in funktionstüchtigen Code umzuwandeln.

Zur Organisation von Aufgaben wird hier eine Methode namens Kanban verwendet. Im Kanban-board werden Aufgaben in 5 verschiedene Spalten aufgeteilt und während des Bearbeitens in die entsprechenden Spalten verschoben.

HTML dient dazu, textbasierte Inhalte eines Webdokumentes zu strukturieren. Dieses Projekt besteht aus 5 separaten Seiten und wurde mittels CSS gestaltet, dabei half die zuvor erwähnte Vorlage aus Figma und erleichterte den Gestaltungsprozess.

JavaScript ist eine Programmiersprache, die Programmierer verwenden, um interaktive Webseiten zu erstellen. Die Speicherung und Überprüfung von Wörtern in Eingabefeldern wurden mithilfe von JavaScript umgesetzt.

Bei GitHub handelt es sich um einen Dienst, der es Programmierern und Entwicklern ermöglicht, den Computercode für ihre Projekte freizugeben. Man kann den Dienst als eine Art Cloud für Computercodes bezeichnen.

Mit diesen oben genannten Mitteln wurde dieses Projekt erstellt.

  1. Die Gestaltung und ideen sammlung in Figma

  2. Organisation von Aufgaben wurde in Kanban verwaltet

  3. HTML und CSS wurde benutzt um das Aussehen der webseite anzupassen

  4. Javascript diente zur Programmierung der Funktion auf der Webseite

  5. Zur Verwaltung des Programmiercodes wurde Git und GitHub verwendet
